# Skill Map

Skills are reusable instructions stored in `AIOS/skills/` so they remain portable between AI tools.

## Available skills

- **Daily Brief** — summarize recent vault activity and surface open work. Trigger: “run my daily brief.”
- **Chronicler** — save or save-and-summarize an AI conversation. Trigger: “save this conversation.”
- **Format Rough Notes** — clean and route notes from `Rough Notes/`. Trigger: “format my rough notes.”
- **Format Working Note** — turn project thinking into a current, actionable working note. Trigger: “format this as a working note.”

When a skill is named, read its complete file before using it.
