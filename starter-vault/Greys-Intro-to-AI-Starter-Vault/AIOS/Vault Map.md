# Vault Map

This vault uses ACE plus an AI operating layer.

- **Atlas/** — durable knowledge, concepts, and references.
- **Calendar/** — daily notes, meeting notes, logs, and Morning Briefs.
- **Efforts/** — active projects and outcome-driven work. Each project can have a Project Home and a current working note.
- **Rough Notes/** — quick capture inbox. Process it only when asked.
- **AIOS/** — personal AI context, maps, reusable skills, and saved conversations.
- **Templates/** — reusable Obsidian note templates.

## Navigation rules

1. Read `AIOS/Me.md` when present; otherwise use `AIOS/StarterMe.md`.
2. Before acting on a project, read its Project Home and the working note marked `status: current`.
3. Use `[[wikilinks]]` when a relationship is useful.
4. Keep generated summaries and conversation history inside `AIOS/` unless the user requests a project, calendar, or knowledge note.
5. Ask before deleting or overwriting existing notes unless a named skill explicitly requires it.
