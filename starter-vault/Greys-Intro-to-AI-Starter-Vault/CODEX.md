# Instructions for Codex

When working in this vault:

1. Read `AIOS/Me.md` if it exists. Otherwise read `AIOS/StarterMe.md`.
2. Read `AIOS/Vault Map.md`, `AIOS/Skill Map.md`, and `AIOS/Tag Map.md`.
3. If the user names a skill, read the matching file in `AIOS/skills/` completely before acting.
4. Treat notes marked `status: current` as the live project state; older working notes may be stale.
5. Ask before publishing, sharing, deleting, or transmitting private information.
6. Keep the user's meaning. Do not invent personal facts, activity, owners, or deadlines.

Use Obsidian-compatible Markdown and `[[wikilinks]]`. Confirm the relevant context you read, then complete the user's request.
