# Grey's Intro to AI — Lesson Context

## Goal

Build a portable personal AI system using Claude Cowork and Obsidian, then leave with one useful workflow already planned.

## What the class covers

1. AI as a city: models, harnesses, context, connectors, skills, MCPs, and automations.
2. G-O-L-U-S prompting: Goal, Output, Limits, Understanding, and Style.
3. A Daily Brief skill and scheduled Cowork task.
4. A personal workflow plan for work, school, or personal life.
5. A final Daily Brief that captures progress and unfinished next steps.

## Daily Brief schedule

- Suggested time: weekdays at 7:59 AM.
- Scheduled tasks run when Claude Desktop is open and the computer is awake.
- If a run is missed, the next run should cover the missed window; Monday reviews Friday through Sunday.

## Privacy rule

Before sharing or publishing a site, document, recording, or AI output, check that it contains no private, sensitive, or restricted information.
